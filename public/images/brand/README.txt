BRAND ASSETS — CLIENT TO PROVIDE
=================================
Place the following files in this directory:

logo.png        — Main logo (dark backgrounds)
logo-white.png  — White version of logo
favicon.ico     — Favicon (32x32 or 64x64)

Recommended sizes:
- logo.png / logo-white.png: 400px wide, transparent background PNG
- favicon.ico: Multi-size ICO file (16, 32, 48px)

Until these are provided, the site will show the text fallback
"Comfy Clean Co." in the navbar and footer.
